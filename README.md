# Car Zag

## Rules
- <kbd>&uarr;</kbd> & <kbd>&larr;</kbd> keys to change direction of car and stay on the path
- <kbd>space</kbd> key to fire ammo and shoot down the towers
- Game will end if car goes off track or hits a tower
